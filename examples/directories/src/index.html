Welcome to the Directories Example Site

This example is idential to the Simplest example, but uses a slightly more complex directory structure, with templates and partials in "_templates" and "_partials" directories.

src/
├── _partials/
│   └── navbar.html
├── _templates/
│   └── base.html
└── index.html

The names of these directories are arbitrary; use whatever names you prefer.

Inside HTML files, the templates and partials are referenced relative to the input directory, as in "_partials/navbar.html" and "_templates/base.html".
